# Hedger — Site Institucional (Multi-página)
Páginas: index.html, sobre.html, servicos.html, como-atuamos.html, contato.html
Stack: HTML5, CSS3, Bootstrap 5, AOS e JavaScript.

Abra `index.html` para navegar. Para publicar, envie tudo para Netlify/Vercel/GitHub Pages.
Formulário de contato é demonstrativo; podemos integrar com EmailJS, Formspree ou API própria.
